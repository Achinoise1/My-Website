# 可变对象和不可变对象

核心区别：**对象创建后，其“内部数据”是否可以被修改。**

## 不可变对象

一旦创建，**值不能被改变**。“修改”实际是创建了一个新对象。

常见不可变类型：

* `int`
* `float`
* `bool`
* `str`
* `tuple`
* `frozenset`
* `bytes`


### 示例：修改整数

```python
a = 10
print(id(a))    # 2762863215120

a = 20
print(id(a))    # 2762863215440
```

两次 `id()` 不一样，说明`10` 是一个对象，`20` 是另一个对象；`a = 20` 不是修改 10，而是让 a 指向新对象 20

本质：不可变对象“修改”= 创建新对象 + 改变引用


### 示例：修改字符串

```python
s = "hello"
s += " world"
```

不是在原字符串上追加，而是新建 "hello world"，让 s 指向它


## 可变对象

创建后，**可以直接修改内部数据，而不会创建新对象。**

### 常见可变类型：

* `list`
* `dict`
* `set`
* `bytearray`
* 自定义 class（默认是可变的）


### 示例：修改列表

```python
lst = [1, 2, 3]
print(id(lst))      # 2818107332416

lst.append(4)
print(id(lst))      # 2818107332416
```

`id()` 不变。

说明列表是在**原对象**上修改的


### 补充说明

Python 变量本质是 **变量名 = 对象的引用**

```
a = [1,2,3]
b = a
```

此时：

```
a → list对象
b ↗
```

修改 a 会影响 b。

```python
a.append(4)
print(b)  # [1,2,3,4]
```

因为它们指向同一个对象。

## 总结

```
变量名 → 引用 → 对象
```

* 不可变对象：修改 = 新建对象
* 可变对象：修改 = 原地修改
