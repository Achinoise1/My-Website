# 深拷贝和浅拷贝

## 赋值（不是拷贝）

```python
a = [1,2,3]
b = a
```

只是复制引用。

```
a → [1,2,3]
b ↗
```


## 浅拷贝

只复制 **第一层对象**
内部嵌套对象仍然共享

常见方式：

```python
b = a.copy()
b = list(a)
b = a[:]
```


### 示例

```python
a = [[1,2], [3,4]]
b = a.copy()

a[0][0] = 999

print(a)
print(b)
```

输出：

```
[[999, 2], [3, 4]]
[[999, 2], [3, 4]]
```

为什么？

```
a  →  [  ──┐
          │
        [1,2]  (同一个)
        [3,4]  (同一个)
b  →  [  ──┘
```

外层 list 复制了
内层 list 没复制


## 深拷贝

递归复制所有层级对象

使用标准库：

```python
import copy
b = copy.deepcopy(a)
```


### 示例

```python
import copy

a = [[1,2], [3,4]]
b = copy.deepcopy(a)

a[0][0] = 999

print(a)
print(b)
```

输出：

```
[[999, 2], [3, 4]]
[[1, 2], [3, 4]]
```

完全独立。


## 对比总结

| 操作   | 是否新对象 | 嵌套对象是否复制 |
| ------ | ---------- | ---------------- |
| 赋值   | ❌          | ❌                |
| 浅拷贝 | ✅          | ❌                |
| 深拷贝 | ✅          | ✅                |

## 什么时候用深拷贝？

1. 处理嵌套数据结构
2. 多线程/异步数据隔离
3. 需要完全独立副本时
4. 递归算法中保护原数据
...
